Back Up Files

Periodically backing up your files helps get rid of malware.

Periodically backup files like documents, pictures, and videos, to make sure that you don't lose valuable information. Don't back up program files, because these files are where infections like to hide. Back up anything that you see as important or valuable, because you have done this this means that your files are safe from being lost, this will prevent the loss of important files and will give you a fail safe for if your system goes under.
